# Stock-Market-Tracker-Cpp

 Real-Time Stock Market Tracker (C++)

##  Overview
This project is a C++ based real-time stock market tracker that simulates stock price updates and allows users to manage portfolios.

##  Features
- Real-time stock price simulation using multithreading
- Buy and sell stocks
- Portfolio management
- Threshold-based alerts
- User authentication system

##  Tech Stack
- C++
- STL (vector, unordered_map)
- Multithreading (thread, chrono)

##  Concepts Used
- Object-Oriented Programming (OOP)
- Data Structures
- Multithreading
- Memory Management

##  Future Improvements
- Add real-time stock API integration
- Improve UI
- Add database support
- Implement thread safety using mutex

## ▶️ How to Run
1. Compile using g++
2. Run the executable
